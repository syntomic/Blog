# Bluelog

*A clean blog.*

Demo: http://syntomic.top

![Screenshot](blog.png)

## Installation

```
$ git clone https://github.com/syntomic/blog.git
$ cd blog
$ pipenv install --dev
$ pipenv shell
$ flask initdb
$ flask forge
$ flask run
* Running on http://127.0.0.1:5000/
```
